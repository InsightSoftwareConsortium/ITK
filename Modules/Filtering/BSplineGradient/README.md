# BSplineGradient

In-tree ITK module that approximates an image gradient from a BSpline fit
to the image intensities, plus related BSpline scattered-data gradient
filters.

Flagship classes: `itk::BSplineGradientImageFilter`,
`itk::BSplineApproximationGradientImageFilter`, and
`itk::BSplineScatteredDataPointSetToGradientImageFilter`.

## Origin

Ingested from the standalone remote module
[**InsightSoftwareConsortium/ITKBSplineGradient**](https://github.com/InsightSoftwareConsortium/ITKBSplineGradient)
on 2026-06-02 via the v4 ingestion pipeline. The upstream repository
will be archived read-only after this PR merges; it remains reachable
